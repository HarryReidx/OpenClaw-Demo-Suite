# 04. Search To Web Report

This demo shows how the agent fetches live search results and organizes them into a shareable webpage report saved under `demo_outputs/search-reports/`.

## Entry

- Default port: `8104`
- LAN access: `http://<your-ip>:8104/`

## Capabilities

- Pulls the most relevant search results for your topic
- Gets the agent to summarize them into a readable Chinese report
- Automatically stores the report as a webpage that can be previewed or shared

## Run

```powershell
.\launch_all.ps1
```
